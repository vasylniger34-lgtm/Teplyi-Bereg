# Teplyi-Bereg

**«Теплий Берег»** — преміальний оздоровчий комплекс відпочинку в селі Бистриця-Гірська, Львівська область. 

Сайт розроблено з використанням сучасних технологій веб-дизайну:
* Фонове відео на головному екрані.
* Преміальна темна кольорова гама з золотими акцентами.
* Повністю кастомні векторні іконки (без стандартних емодзі).
* Динамічна галерея з автозапуском відео та лайтбоксом.
* Інтерактивна карта Leaflet у темній темі з пульсуючим маркером.
* Інтерактивний калькулятор вартості з інтеграцією з месенджерами для швидкого замовлення.
* Повна мобільна адаптація та телефонна інтеграція.

## Розробка та запуск локально

1. Встановіть залежності:
   ```bash
   npm install
   ```

2. Запустіть сервер для розробки:
   ```bash
   npm run dev
   ```

3. Зберіть проект для продакшену:
   ```bash
   npm run build
   ```
